El párrafo PROGRAM-ID en IBM COBOL permite extensiones que no están en la norma ANSI 85 COBOL estándar:

Extensiones IBM permitidas:

  • Nombres de hasta 160 caracteres (norma 85: 30 caracteres)
  • Puede comenzar con subrayado (_)
  • Puede ser un literal alfanumérico entre comillas
  • ID es abreviatura válida para IDENTIFICATION
  • El punto después del nombre es opcional

Código no conforme:

IDENTIFICATION DIVISION.
PROGRAM-ID. _.
      * Nombre inválido: solo un subrayado

PROGRAM-ID. PROGRAMA-CON-NOMBRE-MUY-LARGO-QUE-EXCEDE-LOS-160-CARACTERES-PERMITIDOS-POR-IBM-COBOL-Y-DEBERIA-SER-RECHAZADO-PORQUE-ES-DEMASIADO-EXTENSO-PARA-SER-PROCESADO-CORRECTAMENTE-POR-EL-COMPILADOR-Y-PUEDE-CAUSAR-PROBLEMAS.
      * Excede 160 caracteres

Código conforme:

IDENTIFICATION DIVISION.
PROGRAM-ID. MI-PROGRAMA.

ID DIVISION.
PROGRAM-ID. _PROGRAMA-CON-SUBRAYADO.

PROGRAM-ID. "PROGRAMA-LITERAL".
